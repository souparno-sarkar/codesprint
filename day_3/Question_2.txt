Problem Statement
Kavya wanted to decorate her hostel room wall with a creative alphabet design for the college fest.
This time, she decided to create a beautiful alphabet pattern using only uppercase English 
letters and stars (*).
The pattern rules are:
Print alphabets (A, B, C...) on both diagonals.
The alphabet printed depends on the row number.
Print * in all remaining positions.
Help Kavya print this stylish alphabet cross pattern.
Input Format
A single integer N — size of the pattern.
Output Format
Print the required alphabet pattern.
Example

Input
N = 5

Output
A * * * A

* B * B *

* * C * *

* D * D *

E * * * E

Constraints
1 ≤ n ≤26

Hints (2)
1
Hint 1: Use one loop to move through rows and another loop to print each column of the pattern.

2
Hint 2: Think about the positions where the characters should appear to form an X shape.
