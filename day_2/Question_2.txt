Problem Statement
Soumya is the treasurer of her college club and handles all reimbursements. 
Every time a member spends money, they submit the amount, and Soumya adds it to a running record of 
n transactions — some are credits (+), and some are debits (-). At the end of the month, 
she needs to find the second largest transaction amount for the audit report because the largest 
one is always the annual fest and already accounted for.

Soumya wonders: What is the next biggest transaction she should be looking at?
Can you help Soumya find the answer before the auditors arrive?

Input Format

1. The first line contains an integer N — the number of transactions.
2. The second line contains N space-separated integers representing each transaction amount.
Output Format


Print a single integer — the second largest distinct value.
Example
Input
N = 5

Amounts = [100, 200, 150, 200, 50]

Output
150

Explanation: Largest is 200, second largest distinct value is 150.

Constraints
2 ≤ N ≤ 10^5
-10^4 ≤ amounts[i] ≤ 10^4
At least two distinct values are guaranteed.
