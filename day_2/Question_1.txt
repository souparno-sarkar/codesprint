Problem Statement
Nitin is a huge cricket fan and never misses a match. During the IPL season, 
his professor assigns him n batting scores from different matches. Nitin notices that some players 
scored more than the team's average. He wants to find exactly how many players beat the team average 
so he can argue with his friends about who the real MVP is.
Your task is to help Nitin determine how many players scored strictly above the average of all n scores.


Input Format

1. The first line contains an integer N — the number of players.
2. The second line contains N space-separated integers representing each player's score.
Output Format


Print a single integer — the count of players who scored strictly above the average.
Example
Input
N = 5

Scores = 40 80 60 90 50

Output
2

Explanation: Average = 64. Only 80 and 90 are above it.

Constraints
1 ≤ N ≤ 10^5
0 ≤ scores[i] ≤ 100
